使用Hexo與Github pages搭建個人部落格

發表於 2022-04-24

前言

Github pages是Github提供的網頁代管服務，透過文件來維護更新，不使用資料庫來進行管理。除了Hexo可以作為靜態網站產生器(Static site generator)之外，Hugo和JekyII也可以作為選擇。

靜態網站優點在於

  • 速度快
  • 不需要資料庫
  • 安全性高

Hexo是基於Node.js的框架，使用對於網頁開發者很熟悉的程式語言JavaScript，可以使用Markdown語法建構部落格文章，截至目前有362個主題可供選用，吸引我使用的一部分也是因為Hexo的創建者是台灣人，對於東方語言的支持友善。
