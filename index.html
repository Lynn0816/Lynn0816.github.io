使用Hexo與Github Pages搭建個人部落格

發表於 2022-04-25 更新於 2022-04-29 分類於 Hexo
文章字數： 1.4k 所需閱讀時間 ≈ 1 分鐘

前言

Github pages是Github提供的網頁代管服務，透過文件來維護更新，不使用資料庫來進行管理。除了Hexo可以作為靜態網站產生器(Static Site Generator)之外，Hugo和JekyII也可以作為選擇。

靜態網站優點在於

  • 速度快
  • 不需要資料庫
  • 安全性高

Hexo是基於Node.js的框架，使用對於網頁開發者很熟悉的程式語言JavaScript，支援Markdown語法建構部落格文章，只需要一個指令便可以佈署到Github，截至目前有362個主題可供選用，吸引我的原因也是因為Hexo的創建者是台灣人，對於東方語言的支持友善，擁有廣大的台灣與中國使用者。
